Selection

areContainedByRange(start, end)
areContainedBy(audioQuantum)

Select quanta that begin and end in the specified range.

overlapRange(start, end)
overlap(audioQuantum)

Select quanta that begin or end in the specified range.

endDuringRange(start, end)
endDuring(audioQuantum)

Select quanta that end in the specified range.

startDuringRange(start, end)
startDuring(audioQuantum)

Select quanta that start in the specified range.

containPoint(point)

Select quanta that contain the specified point.

havePitchMax(pitchMax)

Select quanta whose dominant pitch is at the specified index.

havePitchesMax(pitchesMax)

Select quanta whose dominant pitch is at one of the specified indices.

lieImmediatelyBefore(audioQuantum)

Select quanta that end at the start of the specified range.

lieImmediatelyAfter(audioQuantum)

Select quanta that start at the end of the specified range.

fallOnThe(number)

Select quanta that fall on the specified one-indexed position within their group.

overlapEndsOf(audioQuanta)

overlapStartsOf(audioQuanta)

Sorting

confidence

Use the confidence value.

play(analysis.segments.orderedBy(confidence, true));
duration

Use the duration of the segment.

play(analysis.segments.orderedBy(duration));
timbreValue(index)

Use the timbre at the specified index.

play(analysis.segments.orderedBy(timbreValue(0), true));
pitchValue(index)

Use the pitch at the specified index.

play(analysis.segments.orderedBy(pitchValue(1), true));
pitchDistanceFrom(segment)

Use the pitch distance from the specified segment.

timbreDistanceFrom(segment)

Use the timbre distance from the specified segment.

noisiness

Use the noisiness. The noisiness is approximated using the sum of the 12 pitch vectors.
